# Video Game Revenue Analysis Using SQL

SQL analysis identifying top-performing video games and revenue-driving titles.

## Overview
Analyzed video game revenue data to determine which games generate the most revenue and identify the games that contribute most to genre performance.

## Workflow & Tools
- Combined datasets using SQL INNER JOINs
- Used window functions (RANK) to rank games by revenue within each genre
- Analyzed revenue distribution and player engagement indicators

## Key Insights
- Action games generate the majority of total revenue
- A small number of titles dominate revenue within the genre
- High player engagement (reviews and active player base) strongly correlates with revenue performance

## Skills
SQL, JOINs, window functions, data analysis, business insights
